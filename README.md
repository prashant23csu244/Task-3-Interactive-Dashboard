# Task 3: Interactive Dashboard using Power BI

## Objective
Develop an interactive business dashboard using Power BI to analyze sales, profit, customer behavior, and regional performance.

## Tools Used
- Microsoft Power BI
- Microsoft Excel / CSV
- Sample Superstore Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders
- Total Customers
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales by Segment
- Top 10 Products by Sales
- Top 10 Customers by Sales
- Top States by Profit
- Ship Mode Analysis
- Discount by Category

## Key Insights
- Technology generated the highest sales.
- West region achieved the highest profit.
- Consumer segment contributed the largest share of sales.
- Standard Class was the most preferred shipping mode.
- Furniture received the highest average discount.
- Sales showed an upward trend from 2014 to 2017.
